# Holberton Interview

Exercices d'entraînement aux entretiens techniques, réalisés à Holberton School. Chaque dossier contient un problème d'algorithmique indépendant, avec sa solution et ses contraintes propres.

## Projets

| Dossier | Sujet |
|---|---|
| `lockboxes` | Déterminer si toutes les boîtes verrouillées peuvent être ouvertes (parcours de graphe) |
| `log_parsing` | Lecture d'un flux de logs en continu et calcul de métriques au fil de l'eau |
| `minimum_operations` | Nombre minimal d'opérations pour obtenir n caractères H (copier puis coller) |
| `nqueens` | Résolution du problème des N reines par backtracking |
| `pascal_triangle` | Génération du triangle de Pascal jusqu'à n lignes |
| `utf8_validation` | Validation d'un encodage UTF-8 à partir d'une liste d'entiers |

## Langage

Python 3.

## Utilisation

Chaque dossier se lance indépendamment. Exemple :

```bash
cd nqueens
./0-nqueens.py 4
```
